# Changelog

本项目遵循语义化版本格式。

## 0.2.0 - 2026-07-03

- 修复 IAAA 隐藏的“信任此设备”复选框导致自动登录中断的问题。
- 修复页面跳转销毁 JavaScript 执行上下文导致自动登录中断的问题。
- 监控线程每轮读取最新保存的自动登录设置。
- 增加运行日志、启动解释器诊断和无控制台启动异常提示。
- 首次安装增加互斥锁、pip 自修复和失败重试，避免重复双击破坏虚拟环境。

## 0.1.0 - 2026-07-02

- 提供 Windows Tkinter GUI 和一键本地环境安装。
- 支持树洞本科与研究生成绩变化检测。
- 集成 Server酱变化通知。
- 支持 Edge 持久登录会话和可选 DPAPI 加密自动登录。
- 增加环境检测、本地数据卸载、脱敏诊断与单元测试。
